Equipment Utilization Dashboard

Challenge: Inefficiencies arose from the lack of real-time equipment utilization insights, leading to increased costs and delayed maintenance decisions.

Approach:

  • Developed a centralized Power BI dashboard by integrating data from multiple equipment APIs.
  • Designed an ELT pipeline using Azure Data Factory, Data Lake, SQL, and Python for efficient data ingestion, transformation, and processing.
  • Implemented a star schema in a data warehouse to optimize reporting.
  • Enhanced security with RBAC and encryption using Azure Key Vault, and automated alert workflows using Power Automate.

Outcome: Achieved $500K in cost savings in 2024, improved operational efficiency by 35%, and reduced manual reporting efforts by 70%.
